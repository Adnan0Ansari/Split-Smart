# Split-Smart 💳✨

Split-Smart is a high-fidelity, real-time expense sharing application designed with a "Fintech-Premium" aesthetic. It simplifies group finances by automating the "who owes whom" logic using a custom settlement engine, all wrapped in a sleek, modern interface.

## 🚀 The Problem & Solution

### The Problem
* **Mental Load:** Manual tracking of group expenses (trips, rent, dinners) is complex and error-prone.
* **Transaction Bloat:** Traditional tracking leads to unnecessary back-and-forth payments.
* **Social Friction:** Lack of real-time transparency leads to awkward conversations about money.

### The Solution
* **Debt Minimization:** A custom algorithm simplifies the chain of debt, reducing the number of transactions needed to settle up.
* **Single Source of Truth:** Real-time synchronization via Firebase ensures every member sees the same balance instantly.
* **Glassmorphism UI:** A premium, translucent interface that turns financial management from a chore into a sophisticated experience.

---

## ✨ Key Features
- **Smart Settlements:** Automatically calculates the most efficient way to pay back friends.
- **Financial Pulse Dashboard:** View total spending, amounts you are owed, and your debts at a glance.
- **Real-Time Sync:** Instant updates across all devices using Google Firebase.
- **Premium Design:** Modern Glassmorphism UI built with **Tailwind CSS v4** and **Framer Motion**.
- **Group Hubs:** Create dedicated spaces for specific trips or shared living situations.

---

## 🛠️ Technical Stack
- **Frontend:** React.js
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Backend/Auth:** Firebase (Firestore & Auth)
- **Icons:** Lucide-React
